# Rotat Admin Guide

Welcome. This guide explains the **special features of rotat.com** — how each one behaves for customers, and what you as an administrator need to do.

Standard nopCommerce features are not explained here; for those use the official documentation (**Help → Documentation** in the admin menu).

## Sections of this guide

| Section | What you'll find |
|---|---|
| **Website Menu** | Why the Product menu updates itself, and when (rarely) you need to touch it. |
| **Customer Registration** | Phone-number registration with SMS code — how it works and its settings. |
| **Texts and Pages** | Where to change any text: content pages (About Us…) and site labels, in all languages. |
| **Quotes** | The full quote workflow: customer's quote cart → your review → converting to an order. |
| **Credit and Invoices** | The customer credit dashboard, live ERP figures, and credit requests. |
| **Product Search** | The homepage search: by part number and by bearing dimensions. |
| **Account Managers** | Personal contact persons for customers — ERP sync, regions, and assignment. |
| **Shipping** | Land Shipping rates per country, DHL services, and the weight overhead factor. |
| **SDA Orders** | How paid orders are grouped into ERP delivery documents (SDA) automatically. |
| **Card Payments** | The Mastercard gateway — how customers pay by card, settings, and going live. |
| **Pricing Factors** | Percentage price adjustments by country, customer type, category and brand. |

## How to read each section

Every section follows the same simple layout:

1. **What it does** — the behavior in plain words.
2. **Example scenario** — a short story showing how it happens in real life.
3. **What you need to do** — your steps in the admin panel, numbered.
4. **Settings** — only when a section has settings you can change.

## Quick tips that apply everywhere

- After changing anything that customers see, if the website still shows the old version: click the **gear icon** at the top-right of the admin panel to clear the cache, then refresh the page.
- The site runs in three languages — **Arabic, English, French**. Texts edited in one language do not change the others.
